Purpose

This procedure provides an overview and index of all Server Administration Standard Operating Procedures for PowerShell server administration, including firewall management, network configuration, system hardening, and maintenance procedures.

When to Use: - Finding the appropriate server administration procedure - Understanding available server administration tools and scripts - Locating documentation for specific server administration tasks - Reviewing server administration best practices

Quick Reference

Common PowerShell Commands

# Check firewall status
Get-NetFirewallProfile

# Check WireGuard profile
Get-NetAdapter | Where-Object { $_.Name -like "*NYVPNServer*" }

# Check firewall rules
Get-NetFirewallRule | Where-Object { $_.Enabled -eq $true }

# Check network adapters
Get-NetAdapter | Select-Object Name, InterfaceDescription, Status
