Create New SOP Procedure

Metadata

Category: SOP Management Last Updated: 2024-01-15 Author: System Administrator Version: 1.0 Review Date: 2024-04-15 Status: Active

Purpose

This procedure covers using the New-SOP.ps1 script to create new Standard Operating Procedures from templates.

When to Use: - Creating new SOPs - Standardizing SOP format - Quick SOP generation

Prerequisites

Required Access

  • [x] Write access to SOPs directory
  • [x] PowerShell execution permissions

Required Tools

  • [x] PowerShell
  • [x] Script: Scripts\03-SOP-Management\New-SOP.ps1

Procedure Steps

Step 1: Run New-SOP Script

Action:

cd I:\HyperV_PowerScript_Optimazation\Scripts\03-SOP-Management

# Create procedure SOP
.\New-SOP.ps1 -Title "Daily Server Checks" -Category ServerAdministration -Type Procedure

# Create checklist SOP
.\New-SOP.ps1 -Title "Deployment Checklist" -Category MultichartsModels -Type Checklist

Expected Result: - New SOP file created - Opens in VS Code automatically
