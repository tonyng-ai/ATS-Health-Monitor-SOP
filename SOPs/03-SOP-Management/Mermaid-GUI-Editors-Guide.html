True Drag-and-Drop Options

Current State:

There is no fully-featured drag-and-drop GUI editor for Mermaid.js that's open source and production-ready.

Why:

  • Mermaid is designed as text-based
  • Visual editors are complex to build
  • Most tools provide text editing + visual preview

Alternatives:

  1. Draw.io - Full GUI, can export to various formats
  2. Lucidchart - Commercial, has Mermaid export
  3. PlantUML - Text-based like Mermaid, has better GUI tools
