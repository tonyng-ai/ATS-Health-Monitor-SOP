Mermaid.js Quick Start for SOPs

Installation

VS Code Extension (Required)

Install Markdown Preview Enhanced extension: 1. Press Ctrl+Shift+X 2. Search: "Markdown Preview Enhanced" 3. Install (by Yiyi Wang)

This extension renders Mermaid diagrams in Markdown preview.

Quick Usage

1. Create Diagram

Type mermaid-flow + Tab to insert a flowchart template, or write:

```mermaid
flowchart TD
    A[Start] --> B[End]

### 2. Preview

Press `Ctrl+Shift+V` to preview - diagrams render automatically!

---

## Available Snippets

Type these in VS Code and press `Tab`:

| Snippet | What It Creates |
|---------|----------------|
| `mermaid-flow` | Basic flowchart |
| `mermaid-procedure` | Procedure flowchart with verification |
| `mermaid-seq` | Sequence diagram |
| `mermaid-arch` | System architecture |
| `mermaid-state` | State diagram |
| `mermaid-gantt` | Gantt chart |
| `mermaid-pie` | Pie chart |

---

## Common SOP Diagrams

### Procedure Flow

```markdown
```mermaid
flowchart TD
    Start --> Step1 --> Step2 --> Verify{Verify}
    Verify -->|Pass| End
    Verify -->|Fail| Troubleshoot --> Step2

### Network Architecture

```markdown
```mermaid
graph TB
    User --> VPN --> Server
    PublicIP -->|Limited| Server

### Sequence Diagram

```markdown
```mermaid
sequenceDiagram
    User->>Server: Request
    Server-->>User: Response

```

Tips

  1. Test Online First: Use mermaid.live to test diagrams
  2. Keep It Simple: Don't overcrowd diagrams
  3. Use Colors: Make important nodes stand out
  4. Preview Often: Check how it looks in preview
