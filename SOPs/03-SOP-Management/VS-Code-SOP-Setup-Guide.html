VS Code SOP Setup Guide

Quick Setup (5 minutes)

Step 1: Install VS Code Extensions

Open VS Code and install these extensions:

  1. Markdown All in One (by Yu Zhang)
  2. Press Ctrl+Shift+X
  3. Search: "Markdown All in One"

  4. Install

  5. Markdown Preview Enhanced (by Yiyi Wang) ‚≠ê REQUIRED FOR MERMAID

  6. Search: "Markdown Preview Enhanced"
  7. Install

  8. This renders Mermaid diagrams!

  9. Paste Image (by mushan)

  10. Search: "Paste Image"

  11. Install

  12. GitLens (by GitKraken)

  13. Search: "GitLens"
  14. Install

Step 2: Load Snippets

The snippets files are already in .vscode/snippets/: - sop-snippets.code-snippets - SOP templates - mermaid-snippets.code-snippets - Mermaid diagram templates

VS Code will automatically load them when you: - Open the workspace folder in VS Code - Open any .md file - Type in a markdown file

Note: If snippets don't appear, reload VS Code: - Press Ctrl+Shift+P - Type: "Reload Window" - Press Enter

Step 3: Verify Setup

  1. Open any .md file
  2. Type sop-procedure and press Tab
  3. You should see the full SOP template!

  1. Create New SOP: powershell .\New-SOP.ps1 -Title "Daily Checks" -Category ServerAdministration

  3. File will open automatically (if code is in PATH)

Troubleshooting

Snippets Not Working

  1. Check file is .md (markdown)
  2. Reload VS Code: Ctrl+Shift+P > "Reload Window"
  3. Check snippets file exists: .vscode/snippets/sop-snippets.code-snippets

Preview Not Showing

  1. Install "Markdown All in One" extension
  2. Press Ctrl+Shift+V
  3. Check file is valid Markdown

PowerShell Script Not Found

  1. Ensure New-SOP.ps1 is in repo root
  2. Run from repo root directory
  3. Check execution policy: Set-ExecutionPolicy RemoteSigned
