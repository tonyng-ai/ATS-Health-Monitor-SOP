Screenshot Capture Instructions

Windows (Recommended)

  1. Full Screen Capture: Press Win + Shift + S and select "Full screen snip"
  2. Window Capture: Press Win + Shift + S and select "Window snip"
  3. Region Capture: Press Win + Shift + S and select "Rectangular snip"

Using Snipping Tool

# Open Snipping Tool
Start-Process snippingtool

Using PowerShell Screenshot

# Install screenshot module (optional)
Install-Module -Name Screenshot -Force

# Take screenshot
$screenshot = [System.Windows.Forms.Screen]::PrimaryScreen
Add-Type -AssemblyName System.Windows.Forms
[System.Windows.Forms.Screen]::PrimaryScreen | Out-Null

File Format Guidelines

  • Format: PNG (preferred) or JPEG
  • Resolution: 1920x1080 for full dashboard, smaller for partial views
  • Quality: High quality, no compression artifacts
  • Color Mode: RGB
  • File Size: Keep under 500KB per image if possible

Naming Convention

All files should follow this pattern:

ats-health-monitor-[section]-[description].png

Examples: - ats-health-monitor-tab-overview.png - ats-health-monitor-login-page.png - ats-health-monitor-console-startup.png

Checklist

Use this checklist when capturing screenshots:

  • [ ] Dashboard is running at http://localhost:8080
  • [ ] Test data is loaded (use config.test.toml)
  • [ ] Browser is Chrome or Edge (modern UI)
  • [ ] Browser zoom is set to 100%
  • [ ] No personal information visible
  • [ ] No sensitive credentials visible
  • [ ] Timestamps show realistic dates
  • [ ] All UI elements are fully rendered
  • [ ] No browser dev tools visible (unless intentional)

Quick Capture Script

Save this as Capture-Screenshots.ps1:

# Quick screenshot capture reminder script
$screenshots = @(
    "ats-health-monitor-overview.png",
    "ats-health-monitor-console-startup.png",
    "ats-health-monitor-navigation-tabs.png",
    "ats-health-monitor-tab-overview.png",
    "ats-health-monitor-tab-positions.png",
    "ats-health-monitor-tab-fix.png",
    "ats-health-monitor-tab-pro.png",
    "ats-health-monitor-tab-email.png",
    "ats-health-monitor-tab-database.png",
    "ats-health-monitor-tab-atslog.png",
    "ats-health-monitor-tab-stp-orders.png",
    "ats-health-monitor-tab-history.png",
    "ats-health-monitor-tab-health.png",
    "ats-health-monitor-tab-debug.png",
    "ats-health-monitor-tab-setup.png",
    "ats-health-monitor-login-page.png",
    "ats-health-monitor-user-session.png",
    "ats-health-monitor-telegram-alert.png",
    "ats-health-monitor-api-response.png",
    "ats-health-monitor-test-results.png",
    "ats-health-monitor-go-test.png"
)

Write-Host "=== ATS Health Monitor Screenshot Capture ===" -ForegroundColor Cyan
Write-Host ""
Write-Host "Required screenshots:" -ForegroundColor Yellow

$i = 1
foreach ($screenshot in $screenshots) {
    $exists = Test-Path $screenshot
    $status = if ($exists) { "[OK]" } else { "[MISSING]" }
    $color = if ($exists) { "Green" } else { "Red" }
    Write-Host "  $i. $status $screenshot" -ForegroundColor $color
    $i++
}

$existing = (Get-ChildItem -Filter "*.png" -ErrorAction SilentlyContinue).Count
$missing = $screenshots.Count - $existing

Write-Host ""
Write-Host "Total: $($screenshots.Count) required, $existing captured, $missing missing" -ForegroundColor Cyan

After Capturing

  1. Place all PNG files in this images/ directory
  2. Run the checklist script to verify all files exist
  3. Verify images display correctly in the markdown preview
  4. Commit to version control
